# Лабораторная работа №1: Типы данных, память и выражения в С

## Описание работы
Эта лабораторная работа посвящена изучению основ языка программирования C.
В ходе выполнения рассматриваются следующие темы:
- Объявление и инициализация переменных базовых типов (`int`, `float`, `char`, `double`).
- Оператор `sizeof` для определения размера типов данных в памяти.
- Работа с адресами переменных и вывод их значений.
- Арифметические операции: остаток от деления (%), инкремент (++), декремент (--).
- Явное и неявное преобразование типов данных.
- Проблемы точности вычислений с плавающей запятой и переполнение целочисленных типов.
- Ввод данных с клавиатуры и форматированный вывод.
- Базовые алгоритмические задачи.

## Технологии и библиотеки
- **Язык программирования**: C.
- **Компилятор**: `gcc` (GNU Compiler Collection).
- **Используемые библиотеки**:
  - `stdio.h` — для ввода/вывода данных (функции `printf`, `scanf`).
  - `math.h` — для математическмих операций (task9).
  - `stddef.h` — используется `ptrdiff_t` для хранения результатов вычитания указателей.

  ## Задания
1.  **Первая программа с переменными**: Объявление и вывод переменных `int`, `float`, `char`.
2.  **Размеры типов**: Вывод размеров типов `char`, `int`, `float`, `double` с помощью `sizeof`.
3.  **Переменные и память**: Вывод значений и адресов переменных в памяти.
4.  **Остаток от деления**: Считывание двух чисел и вывод остатка от деления первого на второе.
5.  **Инкремент и декремент**: Демонстрация работы префиксных и постфиксных операций `++` и `--`.
6.  **Преобразование типов**: Деление целых чисел и получение.
7.  **Потеря точности**: Демонстрация неточного представления вещественных чисел в памяти (сумма 0.1 и 0.2).
8.  **Переполнение**: Демонстрация эффекта переполнения целочисленной переменной (`int`).
9.  **Вычисление формул**: Расчет значений математических выражений для введенного `x`.
10. **Обмен переменных**: Обмен значениями двух переменных с использованием третьей.
11. **Последняя цифра числа**: Нахождение и вывод последней цифры целого числа.
12. **Разложение числа**: Разделение трехзначного числа на сотни, десятки и единицы.
13. **Формат времени**: Конвертация количества секунд с начала суток в формат `HH:MM:SS`.
14. **Исследование памяти**: Вывод адресов нескольких переменных и анализ расстояния между ними.

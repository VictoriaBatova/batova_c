# Лабораторная работа №2: Структуры в языке C

## Описание работы

Эта лабораторная работа посвящена изучению структурных типов данных в языке программирования C. В ходе выполнения рассматриваются следующие темы:

* Объявление и использование структур (`struct`).
* Создание и работа с массивами структур.
* Вложенные структуры для организации иерархических данных.
* Передача структур в функции по значению и по указателю.
* Использование перечислений (`enum`) для улучшения читаемости кода.
* Использование объединений (`union`) для экономии памяти и хранения данных разных типов.
* Реализация базовых операций со списками данных: добавление, поиск, сортировка.

## Технологии и библиотеки

* **Язык программирования**: C.
* **Компилятор**: `gcc` (GNU Compiler Collection).
* **Используемые библиотеки**:

  * `stdio.h` — для ввода/вывода данных (функции `printf`, `scanf`).

## Задания

1. **Базовая структура**: Создание структуры `Student` с полями "имя", "возраст", "средний балл". Демонстрация объявления переменной, её заполнения (статически и с клавиатуры) и вывода данных на экран.
2. **Массив структур**: Создание массива из 5 структур `Student`. Заполнение массива данными, вывод всех студентов и нахождение студента с максимальным средним баллом.
3. **Вложенные структуры**: Создание структуры `Date` (день, месяц, год) и вложенной структуры `Person` (имя, дата рождения). Заполнение данных и фильтрация людей, родившихся после 2000 года.
4. **Структуры и функции**: Создание структуры `Rectangle` (ширина, высота). Написание функций для вычисления площади и периметра с демонстрацией передачи структуры в функцию по значению и по указателю.
5. **Перечисления (`enum`)**: Создание перечисления `Day` для дней недели. Демонстрация работы с переменной типа `enum` и её использование в конструкции `switch`, а также внутри структуры.
6. **Объединения (`union`)**: Создание объединения `Data` для хранения `int`, `float` и `char`. Демонстрация того, что в объединении одновременно может храниться только одно значение. Создание структуры с объединением и полем-индикатором типа для безопасной работы с данными.
7. **Комплексная работа со структурами**: Создание структуры `Student` с полями: имя, возраст, курс (с использованием `enum`), средний балл. Реализация программы, которая позволяет:

   * Создать массив студентов.
   * Добавлять новых студентов.
   * Выводить список всех студентов.
   * Искать студента по имени.
   * Сортировать студентов по среднему баллу.
   * Весь функционал реализован с использованием отдельных функций и указателей.

## Ответы на контрольные вопросы

### 1. Как объявить структуру в языке C

```c
struct Student {
    char name[50];
    int age;
    float average;
};
```

### 2. Что такое вложенные структуры

Это структуры внутри других структур для группировки данных.

```c
struct Date {
    int day, month, year;
};

struct Person {
    char name[50];
    struct Date birthDate;
};
```

### 3. Как передать структуру в функцию

```c
void func(struct Student s);     // по значению
void func(struct Student *s);    // по указателю
```

### 4. Можно ли создавать массивы структур

```c
struct Student students[5];
```

### 5. Как обратиться к элементу массива структур

```c
students[0].age = 20;
```

### 6. Можно ли использовать enum внутри структуры

```c
enum Course { FIRST, SECOND, THIRD };

struct Student {
    char name[50];
    enum Course course;
};
```

### 7. Можно ли сравнивать структуры оператором ==

Нельзя, нужно сравнивать поля:

```c
if (s1.age == s2.age && strcmp(s1.name, s2.name) == 0) {
    // равны
}
```
